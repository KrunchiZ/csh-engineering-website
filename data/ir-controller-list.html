    • Intellisys SE
       Hidden Left Button
      Hidden left button is located between the Power indicator and the Start Button. To access unit must be in READY TO START or START LOCAL-REMOTE. Press the Load button once then press the hidden left button within 3 seconds. To exit this menu press the Set button twice. Menu shows last shutdown conditions and can be viewed using the Display Select button.

      Hidden Right Button
      Hidden right button is located between the Auto Restart indicator and the Unloaded Stop button. To access unit must be in READY TO START or START LOCAL-REMOTE. Press the Load-Unload button once then press the hidden right button within 3 seconds. To exit press the Display Select button once or wait for 30 seconds. Menu allows changes to the machine configuration.

    • Intellisys SG
       Factory Setpoints
      To access unit must be in READY TO START. Press the MAIN MENU (center) button two times then press the LOAD button then press the UNLOAD button all within 3 seconds. To exit the menu press the STATUS or MAIN MENU buttons or wait 30 seconds.

    • Unigy
       Maintenance Reset
      Press and hold the left and right buttons for 5 seconds. Display will flash indicating the timers are reset. Press and hold the left button for 5 seconds and display will clear. Press the left button to go to the next screen.
